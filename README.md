## About The Project

Repository of the OsuDating.nl frontend.

## Getting Started

### Prerequisites

- yarn
  ```sh
  npm install --global yarn
  ```

### Recommended Tools

- Vue.volar - This is the official Vue extension. (It should notify in VSCode)

### Installation

1. Install Packages
   ```sh
   yarn install
   ```
2. Run Dev Server
   ```sh
   yarn dev
   ```
   Add `--host` to expose to Network.
